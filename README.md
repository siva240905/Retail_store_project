# 🛒 Retail Store Management System

A modern and responsive Retail Store Management System developed to simplify day-to-day retail operations. The application provides an intuitive dashboard for managing products, inventory, billing, sales, and customers efficiently.

## ✨ Features

- 📦 Product Management
- 🏪 Inventory Management
- 💳 POS Billing System
- 📊 Sales Analytics Dashboard
- 👥 Customer Management
- 🔍 Product Search & Filter
- 📈 Real-time Statistics
- 📱 Responsive Design
- ⚡ Fast & User-Friendly Interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Chart.js
- Vercel (Deployment)

## 🚀 Live Demo

https://retail-store-project.vercel.app/

## 🎯 Purpose

This project was developed as a portfolio project to demonstrate frontend development skills, dashboard UI design, and retail management system implementation.
